# Markdown para hablar mejor con la IA

Markdown es una forma simple de ordenar un texto con signos fáciles.
Por ejemplo: `#`, `-`, `**` y `>`.
Sirve para escribir mensajes más claros para una persona y también para una IA.

---

# ¿Por qué usar Markdown?

- Ordena las ideas.
- Hace el mensaje más claro.
- Ayuda a pedir mejor una respuesta.
- Facilita revisar un prompt.

> Si el pedido es claro, la respuesta suele ser mejor.

---

# Signos más usados

| Signo | Para qué sirve | Ejemplo |
|---|---|---|
| `#` | Título | `# Mi tema` |
| `##` | Subtítulo | `## Pasos` |
| `-` | Lista | `- idea` |
| `1.` | Pasos en orden | `1. abrir` |
| `** **` | Resaltar algo importante | `**clave**` |
| `* *` | Dar énfasis | `*nota*` |
| `>` | Aviso o instrucción | `> Responde corto` |
| `` ` ` `` | Mostrar una palabra exacta | `` `Markdown` `` |

---

# Reglas simples

- Usa un título para decir el tema.
- Escribe frases cortas.
- Usa listas si tienes varias instrucciones.
- Pide también el formato de la respuesta.
- Resalta solo lo más importante.

---

# ¿Por qué esto ayuda con la IA?

La IA no adivina lo que quieres.
Necesita instrucciones claras.
Markdown ayuda a separar:

- el tema,
- el público,
- lo que necesitas,
- y el formato de la respuesta.

---

# Ejemplo 1

## Pedido sin orden

Quiero una idea para ahorrar agua en la escuela para niños con pasos y fácil.

## Pedido con Markdown

```markdown
# Tema
Idea para ahorrar agua en la escuela

## Público
Niños y niñas

## Lo que necesito
- Una idea simple
- Tres pasos
- Lenguaje fácil

## Formato
- Respuesta corta
```

La segunda versión es más clara.

# Actividad 1

Convierte este pedido a Markdown:

> Dame una receta saludable para el recreo. Quiero ingredientes, pasos y una explicación fácil.

## Solución modelo

```markdown
# Tema
Receta saludable para el recreo

## Lo que necesito
- Ingredientes
- Pasos
- Explicación fácil

## Formato
- Lista corta
```

# Actividad 2

Completa este modelo:

```markdown
# Tema
...

## Lo que necesito
- ...
- ...

## Formato
- ...
```

## Solución modelo posible

```markdown
# Tema
Cómo cuidar una planta en casa

## Lo que necesito
- Tres consejos
- Lenguaje fácil

## Formato
- Pasos
```

# Actividad 3

¿Cuál pedido está mejor organizado?

## Opción A

Quiero una historia de un perro valiente para niños con final feliz y corta.

## Opción B

```markdown
# Tema
Historia de un perro valiente

## Público
Niños

## Lo que necesito
- Final feliz
- Historia corta
- Lenguaje fácil

## Formato
- Un párrafo
```

## Respuesta esperada

La opción B está mejor organizada.

## ¿Por qué?

- Se entiende más rápido.
- La IA recibe instrucciones claras.
- El resultado puede ser más útil.

# Pequeño truco

Si quieres una mejor respuesta de la IA, puedes pensar así:

1. ¿Cuál es el tema?
2. ¿Para quién es la respuesta?
3. ¿Qué necesito exactamente?
4. ¿Cómo quiero la respuesta?

---

# Mensaje final

Markdown no hace magia.
Pero sí ayuda a poner las ideas en orden.
Cuando el mensaje es más claro, la IA suele responder mejor.
