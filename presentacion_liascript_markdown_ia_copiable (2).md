# Markdown para hablar mejor con la IA

Markdown es una forma simple de organizar texto con signos como `#`, `-` y `**`.
Sirve para que una consigna sea más clara para una persona y también para una IA.

---

# ¿Por qué usar Markdown?

- Ordena ideas.
- Separa partes importantes del mensaje.
- Ayuda a pedir respuestas más claras.
- Facilita revisar y mejorar un prompt.

> Una instrucción clara ayuda a obtener una respuesta más útil.

---

# Signos más usados

| Signo | Uso | Ejemplo |
|---|---|---|
| `#` | Título | `# Mi tema` |
| `##` | Subtítulo | `## Pasos` |
| `-` | Lista | `- idea` |
| `1.` | Orden | `1. abrir` |
| `** **` | Resaltar | `**clave**` |
| `* *` | Énfasis | `*nota*` |
| `>` | Aviso | `> Responde breve` |
| `` ` ` `` | Palabra exacta | `` `Markdown` `` |

Estos signos forman parte de la sintaxis básica de Markdown.

---

# Reglas simples

- Un tema por sección.
- Frases cortas y claras.
- Listas para varias instrucciones.
- Pedir también el formato de respuesta.
- Resaltar solo lo importante.

---

# Ejemplo 1

## Sin Markdown

Quiero una idea para ahorrar agua en la escuela para niños con pasos y fácil.

## Con Markdown

```markdown
# Tema
Idea para ahorrar agua en la escuela

## Público
Niños y niñas

## Lo que necesito
- Una idea simple
- Tres pasos
- Lenguaje fácil

## Formato
- Respuesta corta
```

La segunda versión muestra mejor el objetivo, el público y la forma esperada de respuesta.

# Actividad 1

Convierte este pedido a Markdown:

> Dame una receta saludable para el recreo. Quiero ingredientes, pasos y una explicación fácil.

## Solución modelo

```markdown
# Tema
Receta saludable para el recreo

## Lo que necesito
- Ingredientes
- Pasos
- Explicación fácil

## Formato
- Lista corta
```

---

# Actividad 2

Completa este modelo:

```markdown
# Tema
...

## Lo que necesito
- ...
- ...

## Formato
- ...
```

## Solución modelo posible

```markdown
# Tema
Cómo cuidar una planta en casa

## Lo que necesito
- Tres consejos
- Lenguaje fácil

## Formato
- Pasos
```

Explicación breve:
- El tema está claro.
- La lista dice exactamente qué debe responder la IA.
- El formato pide una respuesta ordenada.

# Actividad 3

¿Cuál pedido está mejor organizado?

## Opción A

Quiero una historia de un perro valiente para niños con final feliz y corta.

## Opción B

```markdown
# Tema
Historia de un perro valiente

## Público
Niños

## Lo que necesito
- Final feliz
- Historia corta
- Lenguaje fácil

## Formato
- Un párrafo
```

## Respuesta esperada

La opción B está mejor organizada porque separa tema, público, pedido y formato.

## Explicación breve

- Se entiende más rápido.
- La IA recibe instrucciones claras.
- El resultado suele ser más útil.

# Mensaje final

Markdown no reemplaza las ideas. Ayuda a ponerlas en orden.
Con una consigna más clara, la IA suele responder de forma más útil y más precisa.
